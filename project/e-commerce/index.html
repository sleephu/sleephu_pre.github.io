E-Commerce Web Platform for Restaurants

  • Java
  • Spring
|
  • Java
  • Spring MVC
  • Spring Security
  • Hibernate
  • MySQL
  • Bootstrap

● A C2C web application was developed for restaurants in Java in Eclipse.

● The features include registration, login, restaurants search feature, menu viewing, order processing and reservation etc.

● Designed and implemented data model in MySQL, and used hibernate as ORM for persistence.

● Technologies used: Spring MVC framework, Hibernate, and Spring Security were used. jQuery, and Bootstrap framework were used on the client side
