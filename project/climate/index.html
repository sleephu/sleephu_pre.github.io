Climate Change Big Data Analytic System Development

  • Big Data
|
  • Java
  • Data Science
  • Spark
  • Cassandra

● A big data analytics platform was implemented with Spark and Cassandra (Deployed on Amazon AWS).

● Analyzed GSOD Climate data from 1921 - 2013, (Raw data size around 21 GB).

● Parsed data with Python and developed apps in Java for data cleansing and preprocessing, and loaded into Cassandra.

● Implemented ARIMA algorithms with MATLAB ARIMA tool jar package to build prediction model for future climate.

● Implemented k-means algorithms on Apache Spark cluster which provides real-time analytics capabilities to build clustering model to find areas with similar climate condition.

● Built a responsive UI using Spring framework for interactive analysis.

● Technologies used: Spark, Cassandra, Spring framework were used at back-end, Google Chart and Bootstrap framework were used at front-end.
