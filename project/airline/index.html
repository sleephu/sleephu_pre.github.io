Airline On-time Performance Big Data Analytic System Development

  • Big Data
|
  • Hadoop
  • HBase
  • Pig
  • Apache Phoenix
  • Hive
  • Java

● A big data analytics platform was implemented with Hadoop and HBase (Deployed on Amazon AWS).

● Analyzed Airline On-time performance data from 1987 to 2008 with hundred millions of records.

● Set up development environment in EC2: Java Environment setup, Hadoop, HBase, Pig, Apache Phoenix, Hive, Spark, Tomcat installation and configuration.

● Developed 10+ MapReduce jobs, several Pig UDF in Java and 10+ Pig scripts for data cleansing and preprocessing, and stored refined data to HBase.

● Implemented logistic regression algorithm using Spark ML-Lib to predict airline delays.

● Technologies used: Hadoop, HBase, Hive, Apache Phoenix, Pig were used at back-end. D3.js, morris.js, crossfilter.js, and Bootstrap framework were used at front-end.
